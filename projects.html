EECS 183 Final Project

In this group project, we took the Mario theme song and using python to extract the notes and sort the different pitches. From there we were told to input a sentence and tokenize the words so that they correspond to a pitch - the end result was a musical composition of the inputted sentense. We split the tasks up between three people - we used the Spotify APIs to get the music data and create tunes.
